# 机器学习

## 机器学习概括

```
第一类

线性回归与Logistic。案例：电子商务业绩预测

岭回归，Lasso，变量选择技术。从一团乱麻中识别有用维度的技巧

降维技术。案例：业绩综合指标设计

线性分类器，Knn算法，朴素贝叶斯分类器，文本挖掘，案例：智能判断垃圾短信，通过文本挖掘给用户加标签，评论自动分析，用户流失预警

决策树，组合提升算法，bagging和adaboost，随机森林。案例：运营商用户分析

支持向量机，为什么能理解SVM的人凤毛麟角？

人工神经网络，单层感知器，线性神经网络，BP神经网络，基于梯度下降的学习算法，图像压缩和银行用户信用评估

通用逼近器径向基函数神经网络，在新观点下审视PDA和SVM。Hopfield联想记忆型神经网络。案例：字符识别，人脸识别

概率神经网络和信念贝叶斯分类器

聚类，孤立点判别。案例：推荐系统，自动品酒器，作弊识别，社会系统团体识别


第二类

遗传算法，案例：应用遗传算法优化运营商网络

粒子群算法，应用于聚类，在游戏软件和机器人设计中应用于寻径

退火算法，应用于计算量指数级增长的NP完全问题，求解TSP问题

随机神经网络与玻尔兹曼机

自组织神经网络，神经网络进阶，浅层机器学习与神经网络的对应关系

深度学习的前世今生，图像学习，编码器，图像压缩，图像特征析取分析

深层贝叶斯信念网络和受限玻尔兹曼机，应用：自然语言处理

卷积神经网络，案例：手写体识别

半监督学习，大数据时代机器学习可能的主流。海量未标定分类数据，判别垃圾短信息遇到的困难，混合模型与EM算法，co-training

半监督学习，基于图的方法，半监督支持向量机

```
